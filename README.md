# SanojEKSTerraform
from these terraform configuration file you can create EKS cluster
NOTE: you should install kubectl on your workstation
STEP 1:
     git clone "repo url"
STEP 2:
    cd SanojEKSTerraform
STEP 3:
  terraform init
STEP 4:
  terraform validate
STEP 5:
 terraform plan
STEP 6:
 terraform apply --auto-approve
